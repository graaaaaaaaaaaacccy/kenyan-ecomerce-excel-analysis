# Kenyan E-commerce Sales Analysis (Excel)

 Overview
This project analyzes a Kenyan e-commerce dataset to understand sales trends, customer behavior, and product performance.

 Objectives
- Identify top-performing products
- Analyze revenue trends
- Understand regional sales distribution

 Tools Used
- Microsoft Excel (Pivot Tables, Dashboards)

 Process
- Cleaned raw data (duplicates, missing values, inconsistencies)
- Created pivot tables for analysis
- Built an interactive dashboard

 Key Insights
- Certain product categories contributed the majority of revenue
- Sales were concentrated in major urban areas
- Some products had high demand but low availability


- Expand reach in underperforming regions
